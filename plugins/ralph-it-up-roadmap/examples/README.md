# Examples

This directory contains sample outputs from the ralph-it-up-roadmap plugin.

## scopecraft/

Example roadmap artifacts for a hypothetical project. These demonstrate:

- Proper formatting and structure
- Expected level of detail
- How to fill out each template

Use these as reference when reviewing your own generated roadmaps.

## Files

| File | Purpose |
|------|---------|
| `VISION_AND_STAGE_DEFINITION.md` | Product vision and maturity stage |
| `ROADMAP.md` | Phased delivery plan |
| `EPICS_AND_STORIES.md` | Backlog with acceptance criteria |
| `RISKS_AND_DEPENDENCIES.md` | Risk register and dependency map |
| `METRICS_AND_PMF.md` | Success metrics and instrumentation |
| `OPEN_QUESTIONS.md` | Blocking decisions and experiments |
